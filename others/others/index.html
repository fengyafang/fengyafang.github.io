    # 日常笔记

    • white-space 父元素位display：flex 时不生效 需要为display:block

    • uniapp中page.json的tabbar路径名必须跟pages中的一样 两个属性中都要写

    • uniapp tabBar 切换时页面中的组件不更新 可以随便传一个值给子组件 在onShow时更改 子组件中用watch监听this.$props.属性名 从而实现切换时更新子组件

    • 箭头函数的arguments和this一样 在定义时就已经绑定 都指向定义时所在函数的值

      function a (param) {
    let b = ()=>{
  	  console.log(this,arguments)
    }
}

      相当于

      function a (param) {
    let that = this,
  	  context = arguments ;
    let b = function (){
  	  console.log(that,context)
    }
}

      可以结合apply(this,arguments)来封装函数 // apply参数为数组或伪数组 call的参数是一个一个值，需要全部列出来

      arguments.callee指向当前正在执行的函数， 函数名.caller指当前函数的调用者=arguments.callee.caller //严格模式下不可以 不能用es6语法写

    • this 谁调用指向谁 将函数名传递给别的对象不起作用 只指向函数执行时前边的对象 也就是函数名后面加()执行了才算

      var b = {a:2}; b.fn=a.fn; b.fn() //2

      this指向与作用域无关

    • 函数有prototype 可以理解为传承 是将属性传递给new出来的实例的 对象有__proto__ 可以理解为继承 继承来自原型的属性、方法 原型有construcer 指向函数

    • js根对象是Object.prototype， Object是一个构造函数

    • transition 立刻设置后不生效 可以将下一个要设置的transition放到一个很小的延迟后面，但不能太小比如轮播中最后一张图切换到第一张图时transition设为0后立刻切换回来

       ele.style.transition='0s'
 ele.style.transform='translateX(0)'
 settimeout(()=>{
 	ele.style.transition='1s'
 	ele.style.transform='translateX(500px)'
 
 },30)

    • import编译时执行 require等是运行时执行

    • class中使用=写法会将函数注册到在实例上 下面代码中funb,func方法在实例上 funa在原型上

       class foo {
 	funa(){
 		
 	}

    • 正则中[]中的(){}含义会发生改变 只是代表符号本身 不代表组或者范围

      var reg1 = /[a{3}]/  //不表示匹配‘aaa’  表示匹配含有 a, {,3,}任一字符的文本
reg1.test('b{r') //true //含有‘{’  成功
reg1.test('aaa') //true //因为含有a 所以成功 并不是因为三个a 
reg1.test('a') //true

    • [asd]=(a|s|d) 都表示含有asd中任意一个字符

    • [$&]≠($|&) [$&]表示 匹配$或者&字符 ($|&)表示 匹配&字符 或者结尾

    • 1px并不代表屏幕上的一个像素点 retina屏上1px可能代表实际上的4个或更多像素点 缩放也会导致1px所代表的的屏幕像素发生改变

    • width=device-width属性将浏览器视口设置为理想视口 通过document.documentElement.clientWidth可以获取 一般为屏幕宽度 ios为320 安卓各有不同

      (function(doc, win) {
    var docEl = doc.documentElement,
        resizeEvt = 'orientationchange' in window ? 'orientationchange' : 'resize',
        recalc = function() {
            var clientWidth = docEl.clientWidth;
            if (!clientWidth) return;
            docEl.style.fontSize =  (clientWidth / 720) + 'px'; //数字为设计稿宽度 可以除以100 在写css时也缩小100倍就可以了
        };
    if (!doc.addEventListener) return;
    win.addEventListener(resizeEvt, recalc, false);
    doc.addEventListener('DOMContentLoaded', recalc, false);
})(document, window);
